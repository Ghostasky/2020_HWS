> Requirements
- A GTK-based Linux (32/64-bit), Windows (32-64-bit), or macOS (64-bit only) operating system
- A Java 8 Runtime Environmemt (visit http://www.oracle.com/technetwork/java/javase/downloads)

> How to install?
- Simply extract the JEB archive contents to a directory of your choice.

> How to set up and use the floating licensed builds?
Tutorial at https://www.pnfsoftware.com/jeb/manual/floating/

> How to execute?
- On Linux, open a terminal and type: "bash jeb_linux.sh"
- On MacOS, open a terminal and type: "bash jeb_macos.sh"
- On Windows, open a terminal and type: "jeb_wincon.bat"
